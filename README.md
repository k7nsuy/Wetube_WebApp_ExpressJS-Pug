# Router
- / => Home
- /join => Join
- /login => Login
- /search => Search

- /users/:id(user) => See User
- /users/logout => logout User
- /users/edit => Edit profile
- /users/delete => Delete User

- /videos/:id(video) => See video
- /videos/:id/edit => Edit video
- /videos/:id/delete => Delete video
- /videos/upload => Upload video
